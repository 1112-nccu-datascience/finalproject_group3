# [Group3] 預測申請人是否會完成貸款流程
目的是確定潛在客戶簽約貸款的可能性。大多數貸款公司分析申請人的財務歷史來決定是否貸款。即本案目的是開發一個型來預測申請者的信用風險情 況。
